# Структура
```bash
├── {course}
│   ├── {i} - номер задачи
│   │   ├── index.py - решение
│   │   ├── description.md - описание
│   │   ├── solution.md - решение с объяснением
│   │   └── tests - тесты для проверки
│   │       ├── 0 - 0й тест
│   │       ├── 1 - 1й тест
│   │       ├── ...
│   │       ├── answers - ответы
│   │       └── index.py - магия
```
