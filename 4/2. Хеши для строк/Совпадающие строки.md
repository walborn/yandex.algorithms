# Совпадающие строки

## Задача
Дано K строк длины N. Для каждой строки посчитать количество совпадающих с ней

## Идея
Код символа - коэффициент полинома (a[i])

## Пример
```py
s = 'abacababa' # 1 2 1 3 1 2 1 2 1
1 * x**8 + 2 * x**7 + 1 * x**6 + 3 * x**5 + 1 * x**4 + 2 * x**3 + 1 * x**2 + 2 * x + x
```
